# Lux Teleport App

Lux Teleport React UI.

### Quick Install

Clone the project.

```
git clone https://github.com/luxfi/teleport-app
```

Install dependencies and build.

```
npm run-script build
```

Run the DApp.<br>

```
npm run start
```

Open [http://localhost:3000](http://localhost:3000) to view it in the browser.
